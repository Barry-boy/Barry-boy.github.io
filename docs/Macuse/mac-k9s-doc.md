## **什么是 k9s **

K9s 是一个基于终端的 UI，用于与 Kubernetes 集群进行交互。该项目的目的是让您更轻松地在野外导航、观察和管理已部署的应用程序。K9s 不断监视 Kubernetes 的变化，并提供后续命令来与您观察到的资源进行交互。
![](https://pic.imgdb.cn/item/64dde60b661c6c8e54a52efd.jpg)


## **如何安装**
- MacOS
```
 # Via Homebrew
 brew install derailed/k9s/k9s
```
