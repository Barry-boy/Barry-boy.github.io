## **程序和进程的关系**

1. 程序
- 保存在硬盘，光盘等介质的可执行代码和数据
- 静态保存的代码

2. 进程
- 在cpu及内存中运行的程序代码
- 动态执行的代码
- 父子进程: 每一个进程可以创建一个或者多个进程

## **四.查看进程信息**
1. pgrep 命令
作用: 根据特定条件查询PID信息
常用选项:
-l : 显示进程名
-U : 指定特定用户
-t : 指定终端

2. pstree 命令

作用： 以树形结构列出进程信息

## **进程的启动方式**

手工启动:

(1) 前台启动: 用户输入命令，直接执行程序

(2) 后台启动: 在命令行尾加入 `&` 符号

(3) 进程的前后台调度
  
  - ctrl+z 将当前进程挂起，及调入后台并停止执行
